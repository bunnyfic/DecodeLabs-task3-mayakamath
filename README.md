# SQL Data Analysis Project

This project was completed as part of the DecodeLabs Data Analyst Internship Program. The objective was to perform data cleaning, database creation, and SQL-based data analysis on a sales dataset.

## Technologies Used

* Python
* Pandas
* MySQL
* Streamlit

## Project Workflow

1. Data Cleaning and Preprocessing
2. Database Creation and Data Import
3. SQL Query Execution
4. Aggregation and Business Analysis
5. Report Generation

## SQL Concepts Covered

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* COUNT()
* SUM()
* AVG()
* Aliases (AS)

## Outcome

The project demonstrates the ability to transform raw sales data into meaningful business insights using SQL and data analysis techniques.
